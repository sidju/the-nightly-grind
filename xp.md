# XP costs

#rules

Here follows a table mapping each stat that can be bought for XP
to the number of XP it costs. See the [character sheet page](character-sheet.md)
for descriptions of what they all mean.

| Stat                   | XP cost   |
|------------------------|-----------|
| Physical Ability       | 4         |
| Mental Ability         | 4         |
| Organizational Ability | 4         |
| HP                     | 3         |
| Humanity               | 7         |
| Lore                   | 3         |
| Any Influence          | 4         |
| In-clan Discipline     | 6, 8, 12  |
| Out-of-clan Discipline | 8, 10, 14 |

(
In-clan formula is 4 + 2^level
Out-of-clan formula is 6 + 2^level
)


BP Increases with age at ca 25, 50, 75 etc
BP has some downsides but is mostly an advantage.
